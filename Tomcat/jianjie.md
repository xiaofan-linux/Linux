#Tomcat简介

```
Tomcat是Apache 软件基金会（Apache Software Foundation）的Jakarta 项目中的一个核心项目
由Apache、Sun 和其他一些公司及个人共同开发而成。
由于有了Sun 的参与和支持，最新的Servlet 和JSP 规范总是能在Tomcat 中得到体现
Tomcat 5支持最新的Servlet 2.4 和JSP 2.0 规范。
因为Tomcat 技术先进、性能稳定，而且免费，因而深受Java 爱好者的喜爱并得到了部分软件开发商的认可，成为目前比较流行的Web 应用服务器。
```
```
JDK是 Java 语言的软件开发工具包，主要用于移动设备、嵌入式设备上的java应用程序。
JDK是整个java开发的核心，它包含了JAVA的运行环境，JAVA工具和JAVA基础的类库。
```
